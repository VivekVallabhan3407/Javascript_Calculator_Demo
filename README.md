# Javascript_Calculator_Demo
This project is a modern calculator built using HTML, CSS, and JavaScript.
It can perform basic arithmetic operations (addition, subtraction, multiplication, division) and includes several scientific features such as square, cube, roots, exponents, and logarithms.

You can also switch between Light and Dark Mode using the theme toggle button for a better user experience.
